# 宿題2
### 課題文
括弧に対応しよう
例： (3.0 + 4 * (2 − 1)) / 5

### 実装方針
括弧のペアを発見するためには、クエリを前から順に見ていって)を見つけたら(を見つけるまで　)のindexから戻る。そのようにして括弧のペアを見つけ、内側を先に計算させるようにした。
